# manuch-scripts - Manuel's Scripts

A collection of personal utility scripts for system administration, homelab management, networking, and automation.

## Contents

| Folder | Description |
|--------|-------------|
| overlay2_usage | Script to check disk space usage of Docker overlayfs/overlay2 directories |

## Usage

Each script is self-contained. Navigate to the relevant folder and read the local `README.md` (if present) for specific instructions. Most scripts require Python 3.9+.

```bash
python3 script_name.py
```
